# Task Plan: Arcade-First Map Game Redesign

## Goal
Define an arcade-first product direction for Ace Map that exceeds current map quiz experiences in replayability, sensory feedback, and spatial gameplay variety.

## Phases
- [x] Phase 1: Explore current project context
- [x] Phase 2: Research competitor strengths and gaps
- [x] Phase 3: Validate product direction with the user
- [x] Phase 4: Write the design document
- [x] Phase 5: Write the implementation plan
- [x] Phase 6: Deliver planning artifacts

## Key Questions
1. Should the product prioritize addictive game feel or study-oriented outcomes?
2. Which core loop produces the fastest replay habit?
3. How should voice, hints, terrain, and region reveal systems reinforce arcade flow instead of slowing it down?

## Decisions Made
- Focus: Arcade-first replayability over exam-first study flow.
- Core loop: Fast 75-105 second runs with immediate replay.
- Primary input model: Mixed mode, with click-region as the default spine and precision pin as a variation.
- Differentiator: Terrain perception, voice-reactive click feedback, and staged reveal systems.
- Content role: Existing topic pages remain for SEO and discovery, but the product center shifts to a dedicated play-first flow.
